# 버전 4.1.1 업데이트 내역

**릴리즈 날짜**: 2025년 12월 28일

## 🎨 UI/UX 개선

### 홈 화면 카드 최적화
- **텍스트 오버플로우 수정**: 주간 매출, 주문 건수 등 통계 카드에서 숫자가 잘리는 문제 해결
  - 반응형 폰트 크기 적용 (화면 크기에 따라 자동 조정)
  - 카드 패딩 최적화로 공간 효율성 향상
  - 긴 숫자도 카드 안에 완전히 표시되도록 개선

- **Hot Products 카드 개선**: 상품명이 잘리는 문제 수정
  - Flexbox 레이아웃 개선으로 전체 너비 활용
  - 상품명 표시 영역 확대
  - 트렌드 지표가 항상 보이도록 레이아웃 조정

### AI Market Intelligence 섹션 개선
- **종합 리포트 미리보기**: 키워드 표시 개수 증가 (2개 → 3개)
  - 한 줄에 3개 키워드 모두 표시
  - 가로 스크롤 지원으로 긴 키워드도 확인 가능
  
- **제목 및 내용 표시 개선**: 
  - 제목 라인 수 증가 (1줄 → 2줄)
  - 뉴스 콘텐츠 미리보기 라인 수 증가 (1줄 → 2줄)
  - 더 많은 정보를 한눈에 파악 가능

## 🐛 버그 수정

### 캘린더 요일 정렬 수정
- **문제**: 캘린더에서 날짜와 요일이 일치하지 않는 문제 (예: 12월 28일 일요일이 토요일로 표시)
- **해결**: 
  - 월의 첫 날 요일 계산 로직 추가
  - 빈 칸을 자동으로 추가하여 올바른 요일에 날짜 배치
  - 모든 월에서 정확한 요일 표시 보장
